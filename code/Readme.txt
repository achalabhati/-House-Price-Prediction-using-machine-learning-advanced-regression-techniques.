Without_FE.py : contains the code without any feature engineering. Models used are  ridge regression, Kernelized Ridge Regression, Lasso Regression, Feed Forward Neural Network, Support Vector Machines, Partial Least Square, Stochastic gradient decent, Bayesian regression.

With_feature_engineering.py : contains the code with feature engineering. Models used are  ridge regression, Kernelized Ridge Regression, Lasso Regression, Feed Forward Neural Network, Support Vector Machines, Partial Least Square, Stochastic gradient decent, Bayesian regression.

With_boosting.py : contains code of boosting algorithm

Data Set link : https://www.kaggle.com/c/house-prices-advanced-regression-techniques/data
